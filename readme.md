# 🚀 Backend Server - Gestión de Archivos CSV

Este proyecto es un **backend desarrollado en Node.js** que permite la **gestión de archivos CSV**, incluyendo la subida, listado con paginación y eliminación de archivos mediante una API REST. Está diseñado para integrarse fácilmente con aplicaciones frontend como Angular.

---

## 📌 Características

- 📤 Subida de archivos CSV al servidor  
- 📄 Listado de archivos con **paginación**  
- 🗑️ Eliminación de archivos mediante API DELETE  
- ⚡ Manejo eficiente de peticiones HTTP  
- 🔗 Integración sencilla con frontend (Angular u otros)

---

## 🛠️ Tecnologías utilizadas

- Node.js  
- Express  
- Multer  
- Csv-parser
- Cors
- Nodemon
