# 🚀 Backend Server - Gestión de Archivos CSV

Este proyecto es un **backend desarrollado con Node.js y Express** que permite la **gestión de archivos CSV**, incluyendo la subida, listado con paginación y eliminación de archivos mediante una API REST.

Está diseñado para integrarse fácilmente con aplicaciones frontend como **Angular**, siguiendo buenas prácticas de desarrollo moderno.

---

## 📌 Características

- 📤 Subida de archivos CSV al servidor  
- 📄 Listado de archivos con **paginación**  
- 🗑️ Eliminación de archivos mediante **API DELETE**  
- ⚡ Manejo eficiente de peticiones HTTP  
- 🔄 Actualización dinámica de datos  
- 🔗 Integración sencilla con frontend (Angular u otros)

---

## 🛠️ Tecnologías utilizadas

- **Node.js**  
- **Express**  
- **Multer** (gestión de subida de archivos)  
- **csv-parser** (procesamiento de archivos CSV)  
- **cors** (control de acceso)  
- **nodemon** (entorno de desarrollo)
